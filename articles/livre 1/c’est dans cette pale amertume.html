C’est dans cette pâle amertume

C’est dans cette pâle amertume que je prends la plume, Dans cette rude solitude que se déroule l’étude, Dans la pure insouciance que se lit tant de souffrance, Dans l’insignifiante vie que s’inscrit mon insolente poésie.
— Gaël Maignan
